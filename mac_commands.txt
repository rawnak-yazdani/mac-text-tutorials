Adding paths to environment variable:

sudo open .zshrc

export PATH=/Library/PostgreSQL/14/bin/:$PATH       // this is not required if PostgreSQL is installed by brew
PROMPT="%F{cyan}%n %1~ %# %f"

source .zshrc

--------------------------------------------------------

source .bash_profile
